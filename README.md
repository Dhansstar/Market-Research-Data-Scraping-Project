# 📌 Project Overview
Proyek ini bertujuan untuk mengevaluasi potensi bisnis skema dropship produk seblak di platform Tokopedia. Dengan memanfaatkan teknik Web Scraping, Data Preparation, dan Practical Statistics, analisis ini mencoba menjawab apakah tren seblak saat ini didukung oleh data penjualan yang nyata atau hanya sekadar viral sesaat.

Version: GAIAv2.2

Last Update: 2025-05-17

🎯 Objectives
Web Scraping: Mengekstraksi data produk dari minimal 10 halaman pencarian Tokopedia menggunakan BeautifulSoup.

Data Preparation: Melakukan cleaning, penanganan missing values, dan konversi tipe data (misal: "750+" menjadi integer 750).

Business Understanding: Merumuskan Problem Statement menggunakan framework SMART.

Practical Statistics: * Analisis deskriptif (Mean, Median, Std Dev, Skewness, Kurtosis).

Estimasi potensi pendapatan menggunakan Confidence Interval.

Uji Hipotesis perbedaan harga (Jabodetabek vs Luar Jabodetabek).

Analisis korelasi antara harga dan minat pembeli menggunakan SciPy.

🛠️ Tech Stack & Installation
Prerequisites
Pastikan Anda sudah menginstall Python 3.9 atau versi terbaru.

Library yang Digunakan
Scraping: BeautifulSoup4, Requests

Data Processing: Pandas, NumPy

Statistics: SciPy

Visualization: Matplotlib, Seaborn

Cara Install

Bash
pip install beautifulsoup4 requests pandas numpy scipy matplotlib seaborn
📊 Dataset Information

Nama Produk: Nama barang yang dijual.

Penjual: Nama merchant/toko.

Kota Penjual: Lokasi pengiriman.

Banyaknya Terjual: Jumlah produk yang laku (dikonversi ke angka).

Rating Produk: Penilaian rata-rata dari pembeli.

# 📌 Project Overview
This project aims to evaluate the business potential of a seblak dropshipping scheme on the Tokopedia platform. Utilizing Web Scraping, Data Preparation, and Practical Statistics techniques, this analysis attempts to determine whether the current seblak trend is supported by real sales data or is simply a temporary viral phenomenon.

Version: GAIAv2.2

Last Update: 2025-05-17

🎯 Objectives
Web Scraping: Extract product data from at least 10 Tokopedia search pages using BeautifulSoup.

Data Preparation: Perform cleaning, handle missing values, and convert data types (e.g., "750+" to the integer 750).

Business Understanding: Formulate a problem statement using the SMART framework.

Practical Statistics: * Descriptive analysis (Mean, Median, Std Dev, Skewness, Kurtosis).

Estimate potential revenue using Confidence Intervals.

Price difference hypothesis test (Jabodetabek vs. Outside Jabodetabek).

Correlation analysis between price and buyer interest using SciPy.

🛠️ Tech Stack & Installation
Prerequisites
Make sure you have Python 3.9 or later installed.

Libraries Used
Scraping: BeautifulSoup4, Requests

Data Processing: Pandas, NumPy

Statistics: SciPy

Visualization: Matplotlib, Seaborn

Installation Method

Bash
pip install beautifulsoup4 requests pandas numpy scipy matplotlib seaborn
📊 Dataset Information
Data is retrieved in real time from Tokopedia with the following features:

Product Name: Name of the item being sold.

Product Price: Price in Rupiah.

Seller: Merchant/store name.

Seller City: Shipping location.

Quantity Sold: Number of products sold (converted to a number).

Product Rating: Average rating from buyers.

Regional Analysis: Price comparison between sellers in Greater Jakarta (Jabodetabek) and those outside Greater Jakarta (Non-Jabodetabek) to see the effect of location on selling prices.
